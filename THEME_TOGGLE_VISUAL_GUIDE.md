## Color Format (HSL)

All colors use HSL format without the `hsl()` wrapper:
```
Correct:   "222.2 84% 4.9%"
Incorrect: "hsl(222.2, 84%, 4.9%)"
```

This format is compatible with Tailwind CSS and shadcn/ui components.
